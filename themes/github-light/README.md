# GitHub Light

GitHub's default light theme.

![GitHub Light preview](preview@2x.png)

| | |
| --- | --- |
| Type | Light |
| Author | GitBox |
| Version | 1.0.0 |

## Install

1. Download [`theme.json`](theme.json) from this folder, or copy the raw URL below.
2. In GitBox, open **Settings > Appearance > Import**.
3. Choose the file. The theme is added to your gallery and applied right away.

Raw URL:

```
https://raw.githubusercontent.com/gitgusilva/gitbox-themes/main/themes/github-light/theme.json
```

## Palette

| Token | Hex | Role |
| --- | --- | --- |
| `bg` | `#FFFFFF` | Base application background |
| `bgElevated` | `#F6F8FA` | Panels, headers, sidebars |
| `bgOverlay` | `#FFFFFF` | Menus, popovers, modals |
| `surfaceHover` | `#EEF1F4` | Hover background for interactive surfaces |
| `border` | `#D0D7DE` | Default borders and dividers |
| `borderStrong` | `#AFB8C1` | Emphasized borders |
| `textStrong` | `#1F2328` | Headings and high-emphasis text |
| `text` | `#24292F` | Primary text |
| `textMuted` | `#656D76` | Secondary and muted text |
| `accent` | `#0969DA` | Primary action and selection |
| `accentHover` | `#2178E0` | Accent hover state |
| `accentFg` | `#FFFFFF` | Foreground drawn on top of the accent |
| `added` | `#1A7F37` | Git added / incoming |
| `removed` | `#CF222E` | Git removed |
| `modified` | `#0969DA` | Git modified / current |
