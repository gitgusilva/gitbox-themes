## Install

1. Download [`theme.json`](theme.json) from this folder, or copy the raw URL below.
2. In GitBox, open **Settings > Appearance > Import**.
3. Choose the file. The theme is added to your gallery and applied right away.

Raw URL:

```
https://raw.githubusercontent.com/gitgusilva/gitbox-themes/main/themes/monokai/theme.json
```

## Palette

| Token | Hex | Role |
| --- | --- | --- |
| `bg` | `#272822` | Base application background |
| `bgElevated` | `#1E1F1C` | Panels, headers, sidebars |
| `bgOverlay` | `#34352F` | Menus, popovers, modals |
| `surfaceHover` | `#3E3D32` | Hover background for interactive surfaces |
| `border` | `#1E1F1C` | Default borders and dividers |
| `borderStrong` | `#49483E` | Emphasized borders |
| `textStrong` | `#FFFFFF` | Headings and high-emphasis text |
| `text` | `#F8F8F2` | Primary text |
| `textMuted` | `#75715E` | Secondary and muted text |
| `accent` | `#66D9EF` | Primary action and selection |
| `accentHover` | `#8AE4F2` | Accent hover state |
| `accentFg` | `#272822` | Foreground drawn on top of the accent |
| `added` | `#A6E22E` | Git added / incoming |
| `removed` | `#F92672` | Git removed |
| `modified` | `#66D9EF` | Git modified / current |
