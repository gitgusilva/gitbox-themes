## Install

1. Download [`theme.json`](theme.json) from this folder, or copy the raw URL below.
2. In GitBox, open **Settings > Appearance > Import**.
3. Choose the file. The theme is added to your gallery and applied right away.

Raw URL:

```
https://raw.githubusercontent.com/gitgusilva/gitbox-themes/main/themes/nord/theme.json
```

## Palette

| --- | --- | --- |
| `bg` | `#2E3440` | Base application background |

| `surfaceHover` | `#4C566A` | Hover background for interactive surfaces |
